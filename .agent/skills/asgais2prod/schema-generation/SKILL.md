# PostgreSQL Schema Generation

**Part of ASG AI S2PROD by [AIShift](https://aishift.dev/)**

## Goal

Generate optimized PostgreSQL database schema with proper indexing, constraints, and **idempotent seed scripts** that never delete existing data.

## âš ï¸ CRITICAL RULES

### Rule 1: ALWAYS Create Database First
```sql
-- database/create-database.sql (run FIRST)
CREATE DATABASE your_app_db;
```

### Rule 2: ALWAYS Grant Permissions
```sql
-- At END of schema.sql
GRANT ALL PRIVILEGES ON ALL TABLES IN SCHEMA public TO your_user;
GRANT ALL PRIVILEGES ON ALL SEQUENCES IN SCHEMA public TO your_user;
```

### Rule 3: SEED SCRIPTS MUST BE IDEMPOTENT

**Seed scripts run on every app restart. They MUST NOT delete existing data!**

```sql
-- âŒ NEVER DO THIS - Deletes all data!
TRUNCATE TABLE users CASCADE;
DELETE FROM users;

-- âœ… ALWAYS USE ON CONFLICT DO NOTHING
INSERT INTO users (email, name, role)
VALUES ('admin@example.com', 'Admin', 'ADMIN')
ON CONFLICT (email) DO NOTHING;
```

### Rule 4: Use Valid UUIDs Only
```sql
-- âœ… GOOD: Use gen_random_uuid()
INSERT INTO users (id) VALUES (gen_random_uuid());

-- âŒ BAD: Invalid hex character 'g'
-- INSERT INTO users (id) VALUES ('b1ffcd00-0d1c-5fg9-...');
```

### Step 2: Generate Schema File

Create `database/schema.sql`:

```sql
-- ================================================
-- Production Database Schema
-- Generated by Google Antigravity
-- ================================================

-- Enable required extensions
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";
CREATE EXTENSION IF NOT EXISTS "pgcrypto";

-- ================================================
-- USERS TABLE
-- ================================================
CREATE TABLE IF NOT EXISTS users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    full_name VARCHAR(255) NOT NULL,
    role VARCHAR(50) NOT NULL DEFAULT 'USER',
    metadata JSONB DEFAULT '{}',
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    
    CONSTRAINT users_role_check CHECK (role IN ('ADMIN', 'COUNSEL', 'REVIEWER', 'USER'))
);

-- ================================================
-- AUTO-UPDATE updated_at
-- ================================================
CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = CURRENT_TIMESTAMP;
    RETURN NEW;
END;
$$ language 'plpgsql';

CREATE TRIGGER update_users_updated_at
    BEFORE UPDATE ON users
    FOR EACH ROW
    EXECUTE FUNCTION update_updated_at_column();

-- ================================================
-- GRANT PERMISSIONS (CRITICAL!)
-- ================================================
GRANT ALL PRIVILEGES ON ALL TABLES IN SCHEMA public TO [db_user];
GRANT ALL PRIVILEGES ON ALL SEQUENCES IN SCHEMA public TO [db_user];
```

### Step 3: Generate Indexes

Create `database/indexes.sql`:

```sql
-- ================================================
-- Performance Indexes
-- ================================================

-- Users table indexes
CREATE INDEX IF NOT EXISTS idx_users_email ON users(email);
CREATE INDEX IF NOT EXISTS idx_users_role ON users(role);
CREATE INDEX IF NOT EXISTS idx_users_is_active ON users(is_active) WHERE is_active = true;
CREATE INDEX IF NOT EXISTS idx_users_created_at ON users(created_at DESC);

-- GIN index for JSONB queries
CREATE INDEX IF NOT EXISTS idx_users_metadata ON users USING GIN (metadata);
```

### Step 5: Create Setup Script

Create `database/setup.sh`:

```bash
#!/bin/bash
# Database Setup Script
# Safe to run multiple times

set -e

DB_NAME="${DB_NAME:-[from config]}"
DB_USER="${DB_USER:-[from config]}"
DB_HOST="${DB_HOST:-localhost}"
DB_PORT="${DB_PORT:-5432}"

echo "ðŸš€ Setting up database..."

# Create database (if not exists)
psql -h $DB_HOST -p $DB_PORT -U $DB_USER -d postgres -c "CREATE DATABASE $DB_NAME;" 2>/dev/null || echo "ðŸ“¦ Database exists"

# Run schema
echo "ðŸ“‹ Running schema..."
psql -h $DB_HOST -p $DB_PORT -U $DB_USER -d $DB_NAME -f database/schema.sql

# Run indexes

psql -h $DB_HOST -p $DB_PORT -U $DB_USER -d $DB_NAME -f database/indexes.sql

```

Make it executable:

## Constraints

- ALWAYS use UUID for primary keys
- ALWAYS add `created_at` and `updated_at` timestamps
- ALWAYS create indexes for foreign keys and frequently queried columns
- ALWAYS use `ON CONFLICT DO NOTHING` in seed scripts
- NEVER use TRUNCATE or DELETE in seed scripts
- ALWAYS include permission grants
- Use `CREATE TABLE IF NOT EXISTS` for idempotency
