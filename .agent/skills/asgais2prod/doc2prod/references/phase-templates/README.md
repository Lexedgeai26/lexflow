# Phase Templates

Templates for different project phases based on complexity and type.

## Available Templates

1. **p0-foundation.md** - Core infrastructure, database, authentication
2. **p1-core-features.md** - Primary business logic and features
3. **p2-secondary-features.md** - Additional features and enhancements
4. **p3-advanced-features.md** - Complex integrations and optimizations
5. **p4-qa-deployment.md** - Testing, documentation, and deployment

## Usage

AI should select appropriate phase templates based on:
- Project complexity
- Feature requirements
- Timeline constraints
- Team size

## Customization

Templates are starting points. AI should:
- Adapt to specific project needs
- Add/remove deliverables as needed
- Adjust acceptance criteria
- Modify based on tech stack
